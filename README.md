# This was created for HackX2.0 conducated at SFIT
This needs a Miner.py and Network.py file to be executed before starting the pages
You need to Import the Wallet extension provided in your prefered Chromium Browser (Tested on Edge and Chrome)

You need to turn on the extension and click new wallet and then open the website and it will detect the extension and let you sign in
In Company Management you need to create a new company and add employees
you can now conduct transaction on the system and send it to other users with their address
their ledger will be on the audit log you can get CSV and pdf from there of your transaction history
You can get hash of a file by uploading it on the system and then compare it with the hash you'll get from the end user
MultiStage approval System and Smart Contract
